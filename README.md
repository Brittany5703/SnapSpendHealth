# SnapSpend Health

SnapSpend Health is a Python-based healthcare expense tracking and receipt organization system.

## Features
- Add healthcare expenses
- Automatic healthcare expense categorization
- Monthly spending summaries
- Most frequent provider tracking
- File storage using Python text files

## Technologies Used
- Python
- GitHub
- File Handling
- Modular Programming

## Categories
- Prescriptions
- Doctor Visits
- Insurance
- Therapy
- Emergency Care
- Medical Supplies

## How to Run

python main.py
